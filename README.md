
# Factor v2.0 - 小鲸鱼智能 RAG 平台

Factor 是一个重新定义本地 LLM 交互体验的 Web GUI，专为 Ollama 和私有知识库 (RAG) 构建。

## 🐳 小鲸鱼 v2.0 特性
*   **重构的 RAG 引擎**：支持更复杂的中文分词匹配，检索更精准，并支持显示引用来源。
*   **状态驱动架构**：彻底解决消息流同步问题，确保每一次提问都能得到即时响应。
*   **视觉重塑**：采用微拟物化与极简主义结合的 UI，支持全系统的流畅暗黑模式。
*   **多源融合**：无缝切换 Ollama 本地模型与 DeepSeek 等云端高性能 API。

## 🚀 核心逻辑
1.  **感知**: 小鲸鱼通过 System Prompt 深度理解自己的身份定位。
2.  **检索**: 在生成回答前，系统会自动在挂载的知识库中进行关键词匹配（RAG）。
3.  **引用**: 回答中会明确标注参考了哪些本地文档。

## 🛠️ 安装与运行
```bash
npm install
npm run dev
```

## 🔐 安全与隐私
Factor 坚持 **Privacy First** 原则：
- 文档解析完全在浏览器内存中完成。
- 对话历史仅存储在 LocalStorage。
- RAG 检索逻辑不经过后端，完全离线化（使用本地模型时）。
